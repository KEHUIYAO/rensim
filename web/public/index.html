rensim

The best puyopuyo training app on Android / iOS / Web
スマートフォン・PC で使えるぷよぷよシミュレータです．
or launch web version

とこぷよモード

メイン機能であるとこぷよシミュレーション．
連鎖アニメーションによる連鎖の確認や，ちぎり数などのデータを確認できます．
Undo/Redo もちろんのこと，盤面の保存・読み込みにも対応するほか，
ツモ補正，初手固定設定などを変更することでいろいろなツモパターンを試せます．

エディットモード

ぷよ設置・削除など基本的なエディット機能を備えています．
とこぷよモードとエディットモードの切り替えはいつでも可能で，
とこぷよの途中にエディットを試しとこぷよに戻るなど，自由に検討することができます．

履歴モード

rensim は undo/redo を含めてすべての操作履歴を保存しています．
履歴はツリー表示されており，自由にタイムトラベルすることができます

カスタマイズ

シミュレータのツモの設定やダブルネクストの表示/非表示切り替え，
スキン変更．2P モードなど様々なカスタマイズを搭載しています．
